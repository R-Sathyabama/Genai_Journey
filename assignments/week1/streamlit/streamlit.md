## Objective
To build a secure, interactive personal productivity dashboard using Streamlit.

## What We Tried
- User authentication (login/signup)
- Personal data vault per user
- Smart daily planner
- Activity calendar visualization
- YouTube study tracker
- Personal journaling system

## How It Works
This Streamlit app manages user state using session variables.
User data is securely stored in local directories with hashed passwords.
Multiple modules are integrated into a single dashboard interface.

## How to Run

### Prerequisites
- Python 3.11+

### Install Dependencies
```bash
pip install streamlit pandas streamlit-calendar streamlit-extras openpyxl
streamlit run streamlit_bot.py
```

###  Output
- Web-based interactive dashboard
- User-specific data storage
